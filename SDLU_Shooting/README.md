# 우주를 배경으로 개발되고 있는 3D FPS

서버개발 노동조합의 3 ~ 4차 성과물이 될 예정입니다.

* * * 

### 개발 엔진
- 유니티 (C#)

### 서버 언어
- Javascript

### DB
- MySQL w/ phpMyAdmin

### 통신
- WebSocket, WebSocketSharp
